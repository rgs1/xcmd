tl;dr
~~~~~

XCmd provides convenient parameters parsing on top of cmd.Cmd

Installing
~~~~~~~~~~

From PyPI:

::

    $ pip install xcmd

Or running from the source:

::

    $ cd xcmd
    $ export FROM_SOURCE=1; bin/xcmd-shell
